## How to Use This Form

### Option 1: Print and Sign

1. Print this form
2. Fill in all fields
3. Both parties sign
4. Scan and email copy to client
5. Keep original in project folder

### Option 2: Google Docs / Digital

1. Copy this template into a Google Doc
2. Fill in all fields
3. Client replies with "Approved" in email
4. Note email approval in signature section
5. Save PDF to project folder

### Option 3: Email Approval

For smaller changes ($100 or less), email approval may be sufficient:

1. Send detailed email with scope, cost, and timeline
2. Client replies with explicit approval
3. Screenshot or save email thread
4. Reference email in this form or project notes

**Recommended:** Use formal change order for anything over $100 or that significantly impacts timeline.
