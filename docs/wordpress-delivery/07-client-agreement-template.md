## 7. Warranty and Support

### 30-Day Bug Fix Warranty

**Period:** 30 calendar days from the date the site goes live on Client's domain

**Covered:** Functionality that worked at launch but stopped working without Client modification. Examples:
- Contact form stops receiving submissions
- Page layout breaks on mobile
- Internal links return 404 errors
- Images stop displaying

### Not Covered (Explicit Exclusions)

The following are NOT covered under warranty:

- Content changes or updates (text, images, phone numbers)
- New features or functionality not in original scope
- Client-caused issues (installing plugins, editing code, changing settings)
- Third-party plugin updates that break functionality
- Hosting or server issues (performance, downtime, SSL expiration)
- Email deliverability problems (spam filtering, domain reputation, blacklisting)
- Domain expiration or DNS configuration issues
- Issues caused by Client's hosting provider

### Post-Warranty Support

After the 30-day warranty:
- **Hourly support:** $50/hour (1-hour minimum)
- **Monthly retainer:** $75/month (includes updates, backups, and up to 1 hour of changes)
- **Emergency same-day:** $100 minimum

**By Client:** Client may terminate this agreement with written notice (email acceptable). If terminated:
- Deposit is non-refundable
- Work completed to date will be delivered in current state
- No further work will be performed

**Mutual Agreement:** Both parties may agree to terminate at any time with terms negotiated in good faith.
