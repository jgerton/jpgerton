<task type="auto">
  <name>Task 1: Configure next-aeo for llms.txt generation</name>
  <files>
    package.json
    .env.example
  </files>
  <action>
Update build script to run next-aeo post-build for llms.txt generation:

**package.json build script:**
```json
{
  "scripts": {
    "build": "next build && next-aeo",
    "build:no-aeo": "next build"
  }
}
```

The next-aeo package:
- Analyzes the Next.js build output
- Extracts page content and structure
- Generates llms.txt file in the public directory
- This file helps AI assistants (ChatGPT, Perplexity, Google AI) discover and understand site content

**Update .env.example** (if exists, otherwise create):
```
# Site Configuration
NEXT_PUBLIC_SITE_URL=https://jpgerton.com

# Analytics (optional)
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX

# Convex
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=
```

Note: The llms.txt file will be generated during `bun run build` and placed in the output directory. Next.js will serve it at /llms.txt.
  </action>
  <verify>
- Run `bun run build` - should execute next-aeo after build
- Check for llms.txt in build output
- After deployment, /llms.txt should be accessible
  </verify>
  <done>Build script configured to generate llms.txt via next-aeo post-build.</done>
