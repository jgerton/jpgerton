Create comprehensive starter template documentation covering:

**1. Theme Selection**
- Kadence Theme as default (block editor native, strong free tier, modern stack)
- Rationale: Built for Gutenberg/FSE future, free version handles 90% of needs, near-perfect Lighthouse scores
- Note flexibility to use Astra or GeneratePress when project needs dictate

**2. Editor Approach**
- Block editor only with Kadence Blocks - NO page builders
- Optimized for Jon's build speed and minimal maintenance
- 9/10 clients want "done for them" sites, won't edit themselves

**3. Essential Plugins (install on every project)**
| Plugin | Purpose | Notes |
|--------|---------|-------|
| Kadence Blocks | Page building blocks | 25+ custom blocks, intelligent load technology |
| Wordfence | Security | Firewall + malware scanner + 2FA |
| UpdraftPlus | Backups | Free backs up to Google Drive/Dropbox |
| Rank Math | SEO | Setup wizard under 5 minutes, local SEO features |
| WP Mail SMTP | Email deliverability | Prevents "I never got the lead" disasters |

**4. Contact Form Solution**
- Use Kadence Advanced Form Block (not WPForms Lite)
- Note: Classic Kadence Form Block is being sunset
- Advanced Form Block is included with Kadence Blocks

**5. "Nice Touch" Extras (optional per project)**
| Plugin | Purpose | When to Include |
|--------|---------|-----------------|
| Site Kit by Google | Analytics dashboard | Always - clients can check visitor stats |
| Starter Templates | Pre-built layouts | For faster initial design |

**6. Standard Page Structure**
- 5-page standard: Home, About, Services, Contact, Privacy Policy
- Expandable to 7 pages if client provides testimonials/FAQ content
- "Fame Wall" testimonial display option modeled after portfolio project cards

**7. Installation Checklist**
Quick reference checklist format Jon can follow for each new project setup.

Format the document with clear headers, tables, and actionable checklists. Include links to plugin WordPress.org pages where helpful.
