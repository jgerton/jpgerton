# Milestone v1.0: MVP

**Status:** SHIPPED 2026-02-04
**Phases:** 1-7
**Total Plans:** 39

## Overview

This roadmap transformed Jon Gerton's portfolio vision into a dual-purpose platform serving local business owners seeking $500 WordPress sites and custom development clients. The journey started with infrastructure foundation and public-facing content, layered on admin capabilities for content management, optimized for discoverability through SEO/AEO, and culminated with WordPress delivery documentation and business protection enabling revenue operations. Each phase delivered verifiable user outcomes building toward the core value: local business owners can see Jon's work, understand the $500 offer, and book a call in under 2 minutes.

## Milestone Summary

**Decimal Phases:** None (all integer phases)

**Key Decisions:**

- Next.js 14 App Router with Bun for modern React patterns and fast installs
- Convex over Supabase for better DX, reactive queries, generous free tier
- Convex Auth over Clerk to keep auth in same ecosystem
- Dockerized dev with polling-based hot reload for volume mounts
- Tailwind v4 CSS-first approach with HSL variables
- nuqs for URL state management
- dnd-kit for accessible drag-and-drop
- next-aeo for llms.txt generation
- Kadence Theme as standard WordPress stack
- 11-section contract structure for client protection

**Issues Resolved:**

- Calendly hydration errors fixed with dynamic import and SSR disabled
- Convex array index limitations worked around with client-side filtering
- ESLint v8 chosen over v9 for Next.js compatibility
- Zod v4 message property instead of errorMap for enum validation

**Issues Deferred:**

- OG image creation (specs documented, needs branded design)
- GA4 setup in Vercel (NEXT_PUBLIC_GA_ID env variable)
- noindex removal for production deployment
- CALENDLY_URL extraction to env variable
- Contract placeholders (phone number, state) need Jon's info

**Technical Debt Incurred:**

- 13 tech debt items documented in v1-MILESTONE-AUDIT.md
- All items are configuration/content tasks, zero code blockers
