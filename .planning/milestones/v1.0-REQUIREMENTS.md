## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Infrastructure (12/12)

- [x] **INFRA-01**: Dockerized local development environment (mandatory - no direct host execution)
- [x] **INFRA-02**: Port mapping in 3400-3499 range (3400 for Next.js, 3401 for Convex)
- [x] **INFRA-11**: Docker volume mounts for live code updates (bind mount codebase, not copy into container)
- [x] **INFRA-12**: Docker Compose for orchestrating Next.js and Convex services
- [x] **INFRA-03**: Next.js 14 App Router project structure
- [x] **INFRA-04**: Convex database and backend integration
- [x] **INFRA-05**: Tailwind CSS + shadcn/ui component library
- [x] **INFRA-06**: Vercel deployment configuration (main branch only)
- [x] **INFRA-07**: Dark/light mode toggle with system preference detection
- [x] **INFRA-08**: Git branching strategy (develop branch for work, main for production releases)
- [x] **INFRA-09**: GitHub Actions workflow for PR validation (lint, type-check, build)
- [x] **INFRA-10**: Branch protection rules (require PR and passing checks before merge to main)

### Public Pages (6/6)

- [x] **PAGE-01**: Home page with hero section, project showcase (6 projects), and services CTA
- [x] **PAGE-02**: Projects page with filterable grid of 6 projects
- [x] **PAGE-03**: Individual project detail pages with screenshots, descriptions, tech stacks, and links
- [x] **PAGE-04**: Services page with 3 tiers ($500 WordPress, Custom Web Apps, Team Growth Accelerator)
- [x] **PAGE-05**: Contact page with Calendly embed for $500 bookings and contact form for other inquiries
- [x] **PAGE-06**: About page with background, experience, and philosophy

### Projects Data (3/3)

- [x] **PROJ-01**: Project data model in Convex (name, description, screenshots, tech stack, links, featured flag)
- [x] **PROJ-02**: Each project displays: name, one-line description, screenshot/preview, tech stack tags, live URL, GitHub repo link
- [x] **PROJ-03**: Projects sortable/reorderable via admin dashboard

### Services Content (5/5)

- [x] **SERV-01**: $500 WordPress Sites tier with deliverables (5-7 pages, mobile responsive, contact form, Google Maps, basic SEO)
- [x] **SERV-02**: Custom Web Apps tier with inquiry-based CTA
- [x] **SERV-03**: Team Growth Accelerator tier with workshop description and CTA
- [x] **SERV-04**: Clear pricing display for $500 tier
- [x] **SERV-05**: Calendly booking button specifically for $500 service

### Contact & Conversion (5/5)

- [x] **CONT-01**: Calendly embed for $500 WordPress site bookings
- [x] **CONT-02**: Contact form with 3 fields max (name, email, message) for custom inquiries
- [x] **CONT-03**: Form submissions stored in Convex database
- [x] **CONT-04**: Email notification on form submission (via Resend)
- [x] **CONT-05**: Dual CTA strategy - distinct paths for WordPress clients vs custom work

### SEO & AEO (8/8)

- [x] **SEO-01**: Proper title and meta description per page
- [x] **SEO-02**: Open Graph tags for social sharing
- [x] **SEO-03**: Semantic HTML structure (h1, h2, proper landmarks)
- [x] **SEO-04**: JSON-LD schema markup (LocalBusiness, Service, Person)
- [x] **SEO-05**: FAQ sections with schema markup on Services page
- [x] **SEO-06**: Speakable markup for AI assistant discoverability
- [x] **SEO-07**: llms.txt generation via next-aeo for ChatGPT/Perplexity/Google AI Overviews
- [x] **SEO-08**: Dynamic sitemap generation

### Admin Dashboard (6/6)

- [x] **ADMIN-01**: Protected admin routes (/admin/*)
- [x] **ADMIN-02**: Convex Auth with email/password login
- [x] **ADMIN-03**: Dashboard home showing recent contact submissions and quick stats
- [x] **ADMIN-04**: Projects CRUD (create, read, update, delete)
- [x] **ADMIN-05**: Project reordering/featuring capability
- [x] **ADMIN-06**: Contact submissions list with read/archive functionality

### Analytics (3/3)

- [x] **ANLYT-01**: Vercel Analytics integration
- [x] **ANLYT-02**: Google Analytics 4 setup
- [x] **ANLYT-03**: UTM parameter tracking for Calendly conversions

### Performance (4/4)

- [x] **PERF-01**: Lighthouse performance score > 90 (achieved 100/96/100)
- [x] **PERF-02**: Page load time < 3 seconds
- [x] **PERF-03**: Image optimization via Next.js Image component
- [x] **PERF-04**: Mobile responsive design (tested on iOS and Android)

### WordPress Delivery System (6/6)

- [x] **WPDEL-01**: Starter template documentation (theme + plugins list)
- [x] **WPDEL-02**: Content workflow documentation (intake -> Claude-assisted drafting -> approval)
- [x] **WPDEL-03**: Local staging setup instructions (Local by WP)
- [x] **WPDEL-04**: Handoff checklist (deployment, DNS, credentials transfer)
- [x] **WPDEL-05**: Payment protection documentation (50% upfront, no launch before final payment)
- [x] **WPDEL-06**: Scope control template (what's included, change request process)

### Business Protection (4/4)

- [x] **BIZPROT-01**: Client agreement template (signable contract covering scope, payment, warranty, liability)
- [x] **BIZPROT-02**: Client qualification checklist (go/no-go filter before quoting)
- [x] **BIZPROT-03**: Hosting requirements document (minimum specs for client hosting)
- [x] **BIZPROT-04**: Project termination protocol (clean exit ramp for bad client relationships)
