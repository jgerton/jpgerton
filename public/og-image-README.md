# Open Graph Image Requirements

Create `og-image.jpg` with these specifications:

## Dimensions
- **Size:** 1200 x 630 pixels (standard Open Graph size)
- **Format:** JPEG (optimized for web, < 150KB)

## Content Suggestions
- "Jon Gerton Web Design" text
- Professional color scheme (blues/grays from brand)
- Optional: tagline "$500 WordPress Sites"
- Clean, modern design

## Tools to Create (Free Options)
- **Figma** - figma.com (professional design tool)
- **Canva** - canva.com (templates available)
- **Adobe Express** - adobe.com/express

## Placement
Place the final image at: `public/og-image.jpg`

## How It's Used
- Social media shares (Facebook, Twitter, LinkedIn)
- Messaging apps (iMessage, WhatsApp, Slack)
- Search engine rich results

## Until Created
Social shares will display without a custom image or use fallback text content.

## Related Assets Needed

### Favicon (favicon.ico)
- **Size:** 32x32 or 16x16 pixels
- **Format:** ICO
- **Placement:** `public/favicon.ico`

### Apple Touch Icon (apple-touch-icon.png)
- **Size:** 180x180 pixels
- **Format:** PNG
- **Placement:** `public/apple-touch-icon.png`

### Tool for Favicon Generation
- **realfavicongenerator.net** - Generates all favicon variants from one image
